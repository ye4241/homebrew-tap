# Ye4241 Tap

## How do I install these formulae?

`brew install ye4241/tap/<formula>`

Or `brew tap ye4241/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "ye4241/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
